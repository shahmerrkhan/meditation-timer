# 🧘 Meditation Timer

A free, no-paywall meditation app built with React. No subscriptions, no sign-ups, just sit down and breathe.

**Live App:** [shahmerrkhan.github.io/meditation-timer](https://shahmerrkhan.github.io/meditation-timer)

---

## Features

- Animated breathing guide with inhale, hold, and exhale cues
- Circular progress ring that drains as your session counts down
- Preset durations (5, 10, 15, 20 min) plus a fully custom timer with hours, minutes, and seconds
- Ambient sounds: Rain, Forest, Waves with volume control
- Mood check-in after every session with optional notes
- Session history with a monthly bar graph, streak tracker, and personal stats
- Floating particle effects during active sessions
- Daily inspirational quote
- Works offline as a PWA, installable on mobile from your browser
- All data stored locally, nothing leaves your device

---

## Tech Stack

- React
- CSS3 (animations, SVG, responsive layout)
- localStorage for session persistence
- ZenQuotes API for daily quotes
- Deployed via GitHub Pages

---

## Running Locally

```bash
git clone https://github.com/shahmerrkhan/meditation-timer.git
cd meditation-timer
npm install
npm start
```
